    ✅ ÉTAPE 1 : CRUD Produits En cours

    Objectif : Créer la gestion complète des produits (Créer, Lire, Modifier, Supprimer)

    Fichiers à créer/modifier :

    • models/Product.js - Modèle Produit (déjà existant ?)
    • controllers/productController.js - Contrôleur pour les produits
    • routes/productRoutes.js - Routes pour les produits
    • views/products/ - Dossier pour les vues produits

    Routes à créer :

    Action URL Méthode Fichier vue
    Liste des produits /shop/products GET products/index.ejs
    Voir un produit /shop/products/:id GET products/show.ejs (ou product.ejs existant)
    Formulaire création /shop/admin/products/new GET products/new.ejs
    Créer produit /shop/admin/products POST Redirection vers liste
    Formulaire modification /shop/admin/products/:id/edit GET products/edit.ejs
    Modifier produit /shop/admin/products/:id POST Redirection vers liste
    Supprimer produit /shop/admin/products/:id/delete POST Redirection vers liste

    Structure des données produit :

    {
  id: 1,
  name: "T-shirt en coton bio",
  slug: "t-shirt-coton-bio",
  description: "Un t-shirt confortable...",
  price: 29.99,
  stock: 150,
  categoryId: 1,
  imageUrl: "/static/images/tshirt.jpg",
  createdAt: "2024-01-20T10:00:00.000Z",
  updatedAt: null
}
